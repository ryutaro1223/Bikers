# <Bikersサイト>

## サイト概要
バイク投稿サイト

### サイトテーマ
バイクに興味を持ってもらう・趣味の共有

### テーマを選んだ理由
バイクに乗る人が増えていく中で、若者に興味を持ってもらいたい

### ターゲットユーザ
バイクに興味がある人・好きな人　一般ユーザー向け

### 主な利用シーン
ツーリング先で撮った写真や納車した写真などを気軽に投稿
バイクに関する質問を投稿
